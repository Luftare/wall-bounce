## Host locally for dev

cd edit-levels
npm install
node .

### Editor:
localhost:9999

### Game:
localhost:9999/game

### Python 2 (game only):
python -m SimpleHTTPServer 8888

### Python 3 (game only):
py -m http.server 8888
